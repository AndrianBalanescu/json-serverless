# Usage
<!-- usage -->
```sh-session
$ npm install -g json-serverless
$ jsonsls COMMAND
running command...
$ jsonsls (-v|--version|version)
json-serverless/0.0.6-alpha.10 darwin-x64 node-v12.12.0
$ jsonsls --help [COMMAND]
USAGE
  $ jsonsls COMMAND
...
```

OPTIONS
  --all  see all commands in CLI
